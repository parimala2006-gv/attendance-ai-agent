Attendance AI Agent

AI-powered attendance analytics dashboard built using Python, Streamlit, and Excel.

Features

- Upload attendance Excel files
- Calculate attendance percentage
- Identify low-attendance employees
- Attendance analytics dashboard
- Visual attendance charts
- HR reporting support

Technologies Used

- Python
- Streamlit
- Pandas
- OpenPyXL
- GitHub

Run Locally

pip install streamlit pandas openpyxl
streamlit run app.py
